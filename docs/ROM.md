# Wave Race 64 (USA) — supported ROM

The port expects a **user-supplied** retail ROM. The repository does not contain game assets.

## Supported revision

| Field | Value |
| --- | --- |
| Title | WAVE RACE 64 |
| Version | USA Rev A (V1.1, 60 FPS / rumble) |
| File size | 8,388,608 bytes (8 MiB) |
| SHA-1 | `508DFC2D4CAA42B6F6DE5263D0AED5E44AC7966A` |
| XXH3-64 (launcher) | `0x2B675E2250A604FC` |
| CRC32 | `DD2BB7B6` |
| Internal PC | `0x80046800` (`BOOT_ADDR` from leak `kn_memory.h`) |

Place your ROM at:

```
WaveRace64Recomp/wr64.us.revA.z64
```

The tooling accepts `.n64` byte-swapped images and normalizes them to big-endian `.z64` automatically.

## Verify your dump

```powershell
python tools/rom_fingerprint.py path\to\your\rom.n64
```

## Leak reference

Checked-out Nintendo-tree leak files live in `leak-ref/` (generated by `tools/rcs_checkout.py` from the RCS archive).

Use `files.def` as the canonical original file list. The later `files` snapshot is the trimmed iQue/Linux tree.

## Notes

- Rev A matches the `sound_rel1/WR_rumbleVer` audio tree in the leak.
- Other regional revisions are unsupported until a separate symbols map is authored.
