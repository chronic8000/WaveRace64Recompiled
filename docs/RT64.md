# WR64 graphics (RT64 / F3DEX)

Wave Race 64 uses **F3DEX FIFO** (`gspF3DEX.fifo.o` in the leak `spec`).

This fork inherits the Zelda64Recomp RT64 integration:

- F3DEX display lists route through `src/game/zelda_render.cpp` and RT64 submodules under `lib/rt64`
- Dual framebuffer / Z-buffer addresses are defined in `include/wr64_memory.h`

WR64-specific display list bugs (water, sky, jet ski banks) are debugged against runtime overlay loads from `recomp_overlays.inl`, using leak `tex_data/` as reference only.
