# Wave Race 64: Recompiled

Native PC port of **Wave Race 64** (N64) via [N64: Recompiled](https://github.com/Mr-Wiseguy/N64Recomp) and [RT64](https://github.com/rt64/rt64).

**This repository contains no game assets** — no ROM, textures, audio, or other copyrighted content from the cartridge. You must provide your own **USA Rev A** `.z64` dump to build or play. Unofficial fan project; not affiliated with Nintendo.

Work in progress. See [BUILDING.md](BUILDING.md) to compile.

Currently this boots to 2 boxes of noise. That is as far as I have got. Please give credit to Chronic Engineering if you get this working! https://www.youtube.com/@ChronicEngineering
